Manual de Uso del ERP

Este manual te guiará paso a paso para utilizar las funcionalidades principales del ERP. Sigue el orden recomendado para asegurar que tu sistema esté configurado correctamente y operando sin problemas.

Paso 1: Añadir un Cliente

Antes de realizar ventas, asegúrate de registrar a tus clientes. Para ventas presenciales con clientes rotativos, puedes crear un cliente genérico llamado "Cliente Genérico".

Añadir Cliente

Paso 2: Añadir un Proveedor

Registra a tus proveedores para gestionar tus pedidos y mantener el control sobre los productos que ingresan a tu inventario.

Añadir Proveedor

Paso 3: Añadir una Categoría

Crea categorías para organizar los productos en tu inventario de manera eficiente y mejorar la gestión de tus datos.

Añadir Categoría

Paso 4: Añadir una Bodega

Registra las bodegas donde almacenarás tus productos. Esto permitirá controlar la ubicación y el stock disponible.

Añadir Bodega

Paso 5: Añadir un Producto

Registra los productos que venderás. Asegúrate de asignarles una categoría, proveedor y ubicación en bodega.

Añadir Producto

Paso 6: Realizar un Pedido

En caso de necesitar reabastecimiento, crea un pedido para registrar compras a tus proveedores.

Realizar Pedido

Paso 7: Registrar una Venta

Una vez configurado todo, comienza a registrar tus ventas. Asegúrate de seleccionar al cliente y los productos correspondientes.

Registrar Venta
